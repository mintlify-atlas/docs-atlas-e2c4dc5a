## Welcome to ScooterShare

ScooterShare is a full-stack IoT scooter-sharing application built to run on Raspberry Pi devices, utilizing Python, Flask, and Google Cloud Platform. The platform provides a complete ecosystem for managing electric scooter rentals with real-time tracking, booking management, and maintenance workflows.

## What Makes ScooterShare Different

ScooterShare stands out as an IoT-enabled platform designed specifically for Raspberry Pi deployment, offering:

- **Multi-Role Architecture**: Separate interfaces and permissions for customers, system administrators, and field engineers
- **Real-Time IoT Integration**: Direct integration with Raspberry Pi hardware for scooter lock/unlock operations
- **Data-Driven Insights**: Seaborn-powered visualizations for usage analytics and reporting
- **Secure Authentication**: BCrypt password hashing with Flask-Session management
- **Gamification**: Points-based ranking system to encourage customer engagement

## Quick Example

Here's a simple example of booking a scooter using the ScooterShare API:

```python
import requests

# View available scooters
response = requests.get("http://192.168.0.3:5000/available_scooters")
scooters = response.json()

# Book a scooter
scooter_id = scooters['scooters'][0]['ScooterID']
user_id = 123

response = requests.post(
    f"http://192.168.0.3:5000/book_scooter/{scooter_id}/{user_id}"
)

if response.status_code == 201:
    print("Booking successful! Scooter status updated to 'Occupying'")
    print("100 reward points added to your account")
```

<Tip>
  The platform automatically updates scooter status and awards loyalty points when a booking is made.
</Tip>

## Architecture Overview

The ScooterShare platform uses a **Blueprint-based architecture** with three main modules:

1. **Customer Module** (`customer.py`) - Handles booking, payments, and usage tracking
2. **Admin Module** (`admin.py`) - Provides fleet management and analytics
3. **Engineer Module** (`engineer.py`) - Manages maintenance and repairs

All modules integrate with a centralized MySQL database and utilize Flask-Session for secure, stateful user management.
