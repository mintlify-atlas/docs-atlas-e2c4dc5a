## Overview

ScooterShare uses MySQL as its database backend, with a schema designed to support user management, scooter fleet tracking, bookings, repairs, and usage analytics.

## Database Tables

### users

Stores all user accounts including customers, admins, and engineers.

<ParamField path="UserID" type="INT" required>
  Primary key, auto-incrementing user identifier
</ParamField>

<ParamField path="UserType" type="VARCHAR">
  User role: `Customer`, `System Admin`, or `Engineer`
</ParamField>

<ParamField path="Username" type="VARCHAR" required>
  Unique username for login
</ParamField>

<ParamField path="Password" type="VARCHAR" required>
  Bcrypt-hashed password (stored as UTF-8 string)
</ParamField>

<ParamField path="FirstName" type="VARCHAR">
  User's first name
</ParamField>

<ParamField path="LastName" type="VARCHAR">
  User's last name
</ParamField>

<ParamField path="PhoneNumber" type="VARCHAR">
  Contact phone number
</ParamField>

<ParamField path="Email" type="VARCHAR" required>
  Email address used for login and notifications
</ParamField>

<ParamField path="AvailableMoney" type="DECIMAL(10,2)">
  Current account balance for scooter rentals
</ParamField>

#### Example Query

```python
insert_query = """INSERT INTO users (UserType, Username,
    Password, FirstName, LastName,
    PhoneNumber, Email, AvailableMoney)
    VALUES (%s, %s, %s, %s, %s, %s, %s, %s)"""

select_query = """SELECT Password, UserID, UserType, Username 
                  FROM users WHERE Email= %s"""
```

### repairs

Tracks maintenance and repair requests for scooters.

<ParamField path="RequestID" type="INT" required>
  Primary key, unique repair request identifier
</ParamField>

<ParamField path="ScooterID" type="INT" required>
  Foreign key to scooters table
</ParamField>

<ParamField path="UserID" type="INT" required>
  User who reported the issue
</ParamField>

<ParamField path="RepairStatus" type="VARCHAR">
  Status: `In Progress` or `Complete`
</ParamField>

<ParamField path="Description" type="TEXT">
  Detailed description of the repair issue
</ParamField>

#### Example Queries

```python
# Create repair request
insert_query = """INSERT INTO repairs
    (ScooterID, UserID, RepairStatus, Description)
    VALUES (%s, %s, %s, %s)"""
cursor.execute(insert_query, 
               (scooter_id, user_id, 'In Progress', description))

# Get all repairs
query = "SELECT * FROM repairs"
cursor.execute(query)

# Update repair status
query = "UPDATE repairs SET RepairStatus = %s WHERE RequestID = %s"
cursor.execute(query, (repair_status, request_id))
```

<Note>
  When a repair is created, the scooter status is automatically updated to `To Be Repaired` and email notifications are sent to all engineers.
</Note>

<ParamField path="UserID" type="INT" required>
  Foreign key to users table

</ParamField>

#### Points System

Customers earn 100 points for each scooter booking:

```python
points_to_increment = 100

# Check if user has existing points
select_query = "SELECT points FROM ranking WHERE UserID = %s"
cursor.execute(select_query, (user_id,))
current_points = cursor.fetchone()

if current_points is not None:
    # Update existing points
    new_points = current_points[0] + points_to_increment
    update_query = "UPDATE ranking SET points = %s WHERE UserID = %s"
    cursor.execute(update_query, (new_points, user_id))
else:
    # Insert new ranking entry
    insert_query = "INSERT INTO ranking (UserID, points) VALUES (%s, %s)"
    cursor.execute(insert_query, (user_id, points_to_increment))
```

## Transaction Examples

### Complete Booking Flow

```python
try:
    connection.begin()
    
    # 1. Update scooter status
    update_scooter_query = """UPDATE scooters 
                              SET Status = 'Occupying' 
                              WHERE ScooterID = %s"""
    cursor.execute(update_scooter_query, (scooter_id,))
    
    # 2. Create booking
    insert_booking_query = """INSERT INTO bookings 
        (ScooterID, UserID, BookingTime, BookingStatus, LockStatus)
        VALUES (%s, %s, NOW(), %s, 'Locked')"""
    cursor.execute(insert_booking_query, 
                   (scooter_id, user_id, 'Active'))
    
    # 3. Award points
    update_points_query = """UPDATE ranking SET points = %s 
                            WHERE UserID = %s"""
    cursor.execute(update_points_query, (new_points, user_id))
    
    connection.commit()
except Exception as e:
    connection.rollback()
    raise e
```

### Complete Rental Flow

```python
try:
    # 1. Charge user
    new_balance = current_money - cost_per_time
    update_query = """UPDATE users SET AvailableMoney = %s 
                      WHERE UserID = %s"""
    cursor.execute(update_query, (new_balance, user_id))
    
    # 2. Lock scooter
    update_query = """UPDATE bookings SET LockStatus = 'Locked' 
                      WHERE BookingID = %s"""
    cursor.execute(update_query, (booking_id,))
    
    # 3. Complete booking
    update_query = """UPDATE bookings SET BookingStatus = 'Completed' 
                      WHERE BookingID = %s"""
    cursor.execute(update_query, (booking_id,))
    
    # 4. Make scooter available
    update_query = """UPDATE scooters SET Status = 'Available', 
                      Location = %s WHERE ScooterID = %s"""
    cursor.execute(update_query, (new_location, scooter_id))
    
    # 5. Record usage
    insert_query = """INSERT INTO scooter_usage 
        (ScooterID, UserID, TimeUsed, UsageDate, DistanceCovered) 
        VALUES (%s, %s, %s, DATE(NOW()), %s)"""
    cursor.execute(insert_query, (scooter_id, user_id, 10, 10))
    
    connection.commit()
except Exception as e:
    connection.rollback()
    raise e
```
